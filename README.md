# FBI Crime Time Series Analysis and Forecasting

## Project Overview
This project focuses on analyzing crime incidents using **Exploratory Data Analysis (EDA)** and **Time Series Forecasting**. The aim of this project is to understand patterns in crime data and predict future crime incidents using machine learning techniques.

Crime analysis is important for improving public safety. By analyzing historical crime data, authorities can identify trends and plan preventive measures.

---

## Dataset Description

The dataset contains crime incident records with the following variables:

- YEAR – Year when the crime occurred  
- MONTH – Month when the crime occurred  
- TYPE – Category of crime incident  
- Incident_Counts – Number of crime incidents  

Each row represents a single crime event.

## Project Workflow

The project follows these steps:

1. Import libraries  
2. Load dataset  
3. Data cleaning and preprocessing  
4. Exploratory Data Analysis (EDA)  
5. Visualization of crime patterns  
6. Time Series analysis  
7. Crime forecasting using ARIMA model  
8. Conclusion

---

## Exploratory Data Analysis

Several charts were created to analyze crime patterns including:

- Crime Type Distribution
- Monthly Crime Trend
- Yearly Crime Trend
- Crime Heatmap
- Time Series Trend

These visualizations help understand how crime incidents vary across time.

---

## Time Series Forecasting

To predict future crime incidents, a **Time Series Forecasting Model** was implemented.

Model Used:

ARIMA (AutoRegressive Integrated Moving Average)

The model forecasts crime incidents for future time periods based on historical patterns.

---

## Key Insights

From the analysis, the following insights were observed:

- Certain crime types occur more frequently than others.
- Crime incidents vary across different months.
- Some time periods have higher crime activity.
- Time series forecasting can help predict future crime trends.

---

## Conclusion

This project demonstrates how **data analysis and time series forecasting** can be used to analyze crime data and predict future incidents. The insights from this analysis can help authorities make informed decisions to improve public safety.
